MedEase Physiotherapy - Image Setup Instructions

Instructions for Adding Images

To complete the website setup, you need to add the following images to the images/ directory. You can download similar images from the original website or use appropriate stock photos.

How to get images from the original website:

  1. Visit the original website: https://www.medeasephysio.com/
  2. Right-click on any image and select "Save image as..."
  3. Save the image with the exact filename listed below
  4. Place all images in the images/ folder

Alternative: Use Stock Photos

You can also use similar stock photos from free sources like Unsplash or Pexels. Just make sure to rename them to match the filenames below.

Image Optimization Tips

  • Use JPEG format for photos and PNG for graphics with transparency
  • Optimize images for web (compress to reduce file size)
  • Maintain aspect ratios as specified
  • Use descriptive alt text (already included in HTML)
